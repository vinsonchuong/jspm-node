## Installing
`jspm-node` is available as an
[npm package](https://www.npmjs.com/package/jspm-node).

## Development
### Getting Started
The application requires the following external dependencies:
* Node.js

The rest of the dependencies are handled through:
```bash
npm install
```

Run tests with:
```bash
npm test
```
